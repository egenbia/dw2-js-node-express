Uma aplicação backend construída com **Node.js e Express.js** — demonstrando fundamentos essenciais de desenvolvimento de APIs REST e backend em JavaScript. Ideal para quem quer começar no mundo do desenvolvimento backend com foco em produção.



## Sobre o Projeto

Este projeto é uma aplicação prática de backend utilizando:

- **Node.js** como runtime JavaScript no servidor;
- **Express.js** como framework web minimalista e flexível para criar APIs;

Ele foi desenvolvido com foco em clareza, organização e boas práticas de estrutura de código.



## Funcionalidades

- Estrutura básica de servidor HTTP com Express  
- Endpoints organizados para aprendizado de rotas  
- Separação de responsabilidades modular  
- Preparado para evoluir para APIs REST completas



##  Tecnologias

 - **Node.js**  Ambiente de runtime para JavaScript no backend 
 - **Express.js**  Framework web para construir APIs rápidas e leves 
 - **JavaScript**  Linguagem principal
